## Getting Started

``` bash
# setup
yarn install

# development mode (with lint)
yarn serve

# check code styles
yarn lint

# unit test by jest
yarn test:unit

# build production for deploy (with typescript check)
yarn build

# git commit by commitizen cli
yarn commit

# take a release commit by standard-version
yarn release
```
